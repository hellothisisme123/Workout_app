Common

Background Grid Spacing

This setting is for the spacing between the dots/graphic. When the setting is set to 10, there is 10px between each dot on the grid background. Some values too low will create significant lag on the users machine, so there is a minimum value for this of 15. WARNING: Changing this setting too quickly in a short amount of time may create large amounts of lags or stutters on the webpage. The lag should resolve if given time.

Background Image

This is the specific image that is displayed on the background grid. The default image is the dots that create a grid pattern on the background of each webpage.

Background Image Opacity

This determines the opacity for the background images. 1 is 100% opacity, 0 is 0% opactity.

Background Image Size

This is for the size of that background images. If you increase this value it makes the images larger. Be warned that if the value is too large there may be clipping.

Background Color

This is the background color of the space between the graphic patters. The value is a hex code, or rgb value.
